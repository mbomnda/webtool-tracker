# Webtool Tracker 

A simple request tracker for internal web tools. It is built with ASP.NET Core

## Why I built it 

First .NET project, coming from PHP MVC. I am an Operations Supervisor that is learning the .NET framework, and this webapp mirrors the request-log workflow internal development teams use.

## Stack 

- ASP.NET Core 8 MVC
- EF Core + SQLite
- Razor
- jQuery

## Status Roadmap 

- [x] Project scaffold
